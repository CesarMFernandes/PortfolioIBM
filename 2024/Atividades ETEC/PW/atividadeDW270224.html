Estrutura Basica do HTML

Olá. Este documento esplicará como é montado um site básico em HTML, explicando vários conceitos que serão necessários para dar seus primeiros passos no mundo de desenvolvemento de web.

O que são as tags, e como usá-las?

As tags funcionam como comandos que servem na estrutura de seu site. Elas podem alterar o tamanho de sua fonte, criar listas, adicionar imagens, e muito mais.

Para começar, todas as tags devem estar entre "<>", e boa parte delas vêm em pares. Para colocá-las, simplesmente digite a primeira tag e uma segunda tag do par aparecerá com uma barra antes dela. Por exemplo, as tags h1 e /h1 formam um par, e devem ser usadas em conjunto, além disso, o texto deve ser colocado entre elas.

Porém, algumas tags não têm um par, e após elas deve ser colocado um espaço e uma barra. A tag hr / é um exemplo disto.

Começo e head

No começo de todos os documentos html, deve ser colocada a tag !DOCTYPE html, que declara que este é um documento HTML, embaixo disso, deve ser colocada a tag html e seu par /html, e o resto documento deve ser colocado entre essas duas tags.

O head, ou a cabeça de seu documento é onde você coloca dados que geralmente não são vistos por pessoas que visitam o site. Ao invés disso, essas tags são usadas para dar informações aos navegadores, como Google Chrome e Microsoft Edge. Para denominar esta parte do documento, ela deve ser colocada entre as tags head e /head

Algumas das tags usadas no head do seu documento são:

  • meta
    • Usado para inserir metadados para os navegadores.
  • title; /title
    • Usado para inserir o título que aparecerá na guia do navegador.

Body e final

O body, ou o tronco de seu documento é onde você coloca as informações para os visitantes do site verem, seja esta texto, listas, imagens, etc. Para denominar esta parte do documento, ela deve ser inserida entre as tags body e /body

Algumas das tags usadas no body do seu documento são:

  • h1 e /h1 -> h6 e /h6
    • Usadas no cabeçalho. Vai de 1 até 6, e quanto maior o número, menor o tamanho da fonte.
  • hr /
    • Cria uma linha horizontal.
  • p e /p
      Usado para denominar o começo e o fim de um parágrafo.
  • br /
    • Usado para fazer uma quebra de linha.
  • pre e /pre
    • Usados para inserir um texto pré definido, por exemplo, uma arte ASCII.
  • li e /li
    • Usado em todos os tipos de lista
      • ul e /ul
        • Usados em listas não ordenadas (que não usam números)
      • ol e /ol
        • Usados em listas ordenadas (que usam números)

Para finalizar, não esqueça que seu documento deve terminar com a tag /html e nada deve ser colocado após ela.

Isso é tudo. Você deve ter aprendido o básico de programação de web, porém, boa parte do seu aprendizado acontecerá quando você mesmo praticar fazendo seus próprios sites.
